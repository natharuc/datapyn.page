analise_vendas.dmp
SQL
SELECT 
    c.nome AS cliente,
    COUNT(p.id) AS total_pedidos,
    SUM(p.valor) AS valor_total
FROM clientes c
INNER JOIN pedidos p ON c.id = p.cliente_id
GROUP BY c.nome
ORDER BY valor_total DESC;
resultado disponível como df
Python
# Top 10 clientes em gráfico

Por que escolher o DataPyn?

Desenvolvido para analistas que precisam de produtividade real no dia-a-dia

Blocos Mistos

Escreva SQL e Python no mesmo documento. O resultado do SQL está automaticamente disponível como DataFrame.

Múltiplas Sessões

Cada aba tem sua própria conexão e namespace Python. Compare dados entre ambientes facilmente.

Import Inteligente

Arraste CSV, JSON ou Excel para o editor. Código de importação gerado automaticamente.

Atalhos Poderosos

F5 para SQL, Shift+F5 para Python. Produtividade máxima.

Visualização Integrada

Matplotlib integrado. Crie gráficos diretamente a partir dos resultados das suas queries.

Temas Claro e Escuro

Interface adaptável ao seu estilo. Syntax highlighting otimizado para cada tema.

Início Rápido

Comece a usar o DataPyn em menos de 5 minutos

1

Clone o repositório

git clone https://github.com/natharuc/datapyn.git
cd datapyn
2

Instale as dependências

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
3

Execute o DataPyn

python source/main.py
